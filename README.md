# UniDone - Найкращий менеджер завдань для студентів 

**Менеджер студентських завдань (MVP)**

Проєкт створюється в рамках циклу лабораторних робіт з дисципліни "Управління IT проєктами" для демонстрації повного життєвого циклу розробки ПЗ (SDLC).

## Поточний етап
**Лабораторна робота №2:** Забезпечення якості через багаторівневе тестування.
- Додано базову бізнес-логіку на JavaScript (розрахунок днів до дедлайну).
- Налаштовано середовище Node.js та `npm`.
- Написано Unit-тести за допомогою **Vitest** та згенеровано звіт про покриття коду (Code Coverage).
- Налаштовано наскрізне (E2E) тестування інтерфейсу за допомогою **Playwright**.

**Лабораторна робота №1:** Управління кодом та організація робочого простору в Git.
- Створено базову структуру репозиторію та налаштовано `.gitignore`.
- Опрацьовано стратегію розгалуження та вирішення конфліктів злиття
## Заплановані можливості (MVP)
- Додавання нових навчальних завдань.
- Встановлення дедлайнів та автоматичний розрахунок днів до здачі.
- Відстеження статусу виконання.

## Технології (На даному етапі)
- **Frontend:** HTML5, Vanilla JavaScript
- **Unit Тестування:** Vitest
- **E2E Тестування:** Playwright
- **Контроль версій:** Git / GitHub
